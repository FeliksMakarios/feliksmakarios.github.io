Sebagai dosen tetap di Fakultas AI dan Data Science, saya mengajar berbagai mata kuliah
di kurikulum S1 — mulai dari pengantar komputasi hingga machine learning lanjut dan NLP.
Saya juga terlibat dalam task force kampus dan fakultas untuk quality assurance akademik
dan pengembangan kurikulum.

## Mata kuliah

### Dasar

- **Introduction to Informatics &amp; Computing** — mata kuliah tahun pertama yang mencakup
  computational thinking, dasar pemrograman, dan lanskap disiplin ini.
- **Object-Oriented Modeling** — prinsip OOP dan pemodelan berbasis UML.
- **Database Systems** — pemodelan relasional, SQL, normalisasi, dan pengantar NoSQL.

### Data science &amp; machine learning

- **Introduction to Data Science** — pipeline data science, exploratory analysis, dan
  pemodelan dasar.
- **Advanced Machine Learning** — fondasi deep learning, representation learning, dan
  arsitektur kontemporer terpilih.
- **Data Science Application Development Studio** — mata kuliah bergaya capstone di mana
  mahasiswa membangun dan men-deploy aplikasi data end-to-end.

### Spesialisasi

- **Natural Language Processing** — pendekatan klasik dan neural, dengan fokus praktis
  pada aplikasi bahasa Indonesia menggunakan Hugging Face Transformers.

## Bimbingan

Saya membimbing riset mahasiswa S1 di topik question answering berbasis Transformer, teks
klasifikasi, aspect-based sentiment analysis, dan studi bias / fairness pada NLP
Indonesia. Kalau kamu sedang mencari dosen pembimbing, silakan lihat <a href="/id/research/">bidang riset</a>
untuk topik yang terbuka.

## Pengalaman mengajar sebelumnya

Sebelum di UPH, saya mengajar di Raffless Institute of Higher Education (Jakarta, 2022)
untuk mata kuliah pengantar computational thinking, Python, databases, algoritma,
matematika diskrit, dasar neural network, analisis data, statistika untuk AI, dan sistem
operasi / jaringan. Sebelumnya lagi, saya mengajar dasar komputer di Charis National
Academy (2015–2017) dan merancang silabus ICT tingkat SD.
